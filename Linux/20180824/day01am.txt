			mkdir   -p   a/b/c



		几个常用的命令：
			ls	显示当前目录里的内容（list）
			ls  -a   显示所有的内容（all 包含隐藏文件）
			ls  -l   显示内容的长格式（long，详细信息）
			ls  -h   显示用户易读的格式（15245）
			ls  -lh  选项是可以组合的(-l -h)

			D://Jay/qilixiang
			D://Jay/qilixiang/shuangjiegun.mp3

			ls  参数（路径）


			路径：
				绝对路径（从根目录 / 开始）  /root/Jay
					根目录  /   是Linux系统的最外层的文件夹
					/root
					
				相对路径（以当前目录作为开始）  ls  Jay


				~   当前用户的主目录
				ls ~

				..  上层目录（父目录）
				ls  ../..
				ls  ..

				.   当前目录
				ls .


			pwd   显示当前的路径
				print  working  directory


			cd    切换路径
				change directory
				cd  要切换的目标路径

				cd  /
				pwd
				ls

				cd  ~
				pwd

				cd ..
				pwd

				cd ../..
				pwd
